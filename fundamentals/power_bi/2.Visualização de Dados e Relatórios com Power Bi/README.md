# Análise de Dados com Power BI: Quando usar cada gráfico? 


## Gráficos de Comparação

### 1. Gráfico de Colunas Agrupadas

*   **Descrição:** Exibe dados em colunas verticais, agrupadas por categoria. Útil para comparar múltiplos valores para cada categoria.
*   **Quando usar:**
    *   Comparar valores de diferentes categorias em um único ponto no tempo.
    *   Visualizar a distribuição de dados entre diferentes grupos.
*   **Exemplo:** Comparar as vendas de diferentes produtos por região.

### 2. Gráfico de Colunas Empilhadas

*   **Descrição:** Similar ao gráfico de colunas agrupadas, mas as colunas são empilhadas umas sobre as outras. Mostra a contribuição de cada parte para o todo.
*   **Quando usar:**
    *   Comparar valores totais entre categorias e também a contribuição de cada subcategoria.
    *   Visualizar a composição de um todo.
*   **Exemplo:** Mostrar as vendas totais por região, com a contribuição de cada produto para o total.

### 3. Gráfico de Barras Agrupadas

*   **Descrição:** Exibe dados em barras horizontais, agrupadas por categoria. Similar ao gráfico de colunas agrupadas, mas com orientação horizontal.
*   **Quando usar:**
    *   Comparar valores de diferentes categorias quando os rótulos são longos.
    *   Visualizar a distribuição de dados entre diferentes grupos.
*   **Exemplo:** Comparar as vendas de diferentes produtos por região, quando os nomes dos produtos são longos.

### 4. Gráfico de Barras Empilhadas

*   **Descrição:** Similar ao gráfico de barras agrupadas, mas as barras são empilhadas umas sobre as outras.
*   **Quando usar:**
    *   Comparar valores totais entre categorias e também a contribuição de cada subcategoria, quando os rótulos são longos.
*   **Exemplo:** Mostrar as vendas totais por região, com a contribuição de cada produto para o total, quando os nomes dos produtos são longos.

## Gráficos de Tendência

### 5. Gráfico de Linhas

*   **Descrição:** Exibe dados como uma linha contínua, mostrando a tendência ao longo do tempo.
*   **Quando usar:**
    *   Visualizar tendências ao longo do tempo.
    *   Mostrar a evolução de um valor ao longo de um período.
*   **Exemplo:** Mostrar a receita mensal de uma empresa ao longo de um ano.

### 6. Gráfico de Área

*   **Descrição:** Similar ao gráfico de linhas, mas a área abaixo da linha é preenchida. Enfatiza a magnitude da mudança ao longo do tempo.
*   **Quando usar:**
    *   Visualizar tendências ao longo do tempo e também a magnitude das mudanças.
    *   Mostrar o volume de dados ao longo de um período.
*   **Exemplo:** Mostrar o número de visitantes em um site ao longo de um ano, destacando os períodos de maior tráfego.

### 7. Gráfico de Área Empilhada

*   **Descrição:** Similar ao gráfico de área, mas as áreas são empilhadas umas sobre as outras. Mostra a contribuição de cada parte para o todo ao longo do tempo.
*   **Quando usar:**
    *   Visualizar a evolução de múltiplos valores ao longo do tempo e a contribuição de cada um para o total.
*   **Exemplo:** Mostrar as vendas totais de uma empresa ao longo do tempo, com a contribuição de cada produto para o total.

## Gráficos de Distribuição

### 8. Gráfico de Dispersão (Scatter Plot)

*   **Descrição:** Exibe dados como pontos em um plano cartesiano. Útil para identificar correlações entre duas variáveis.
*   **Quando usar:**
    *   Visualizar a relação entre duas variáveis.
    *   Identificar padrões e outliers.
*   **Exemplo:** Mostrar a relação entre gastos com publicidade e vendas.

### 9. Histograma

*   **Descrição:** Exibe a distribuição de frequência de uma única variável.
*   **Quando usar:**
    *   Visualizar a distribuição de uma única variável.
    *   Identificar a frequência de diferentes valores.
*   **Exemplo:** Mostrar a distribuição de idades em um grupo de pessoas.

## Gráficos de Relação

### 10. Gráfico de Rosca (Donut Chart)

*   **Descrição:** Similar ao gráfico de pizza, mas com um buraco no centro. Mostra a contribuição de cada parte para o todo.
*   **Quando usar:**
    *   Visualizar a proporção de diferentes categorias em um conjunto de dados.
    *   Mostrar a composição de um todo.
*   **Exemplo:** Mostrar a distribuição de clientes por região.

### 11. Gráfico de Pizza

*   **Descrição:** Exibe dados como um círculo dividido em setores, mostrando a contribuição de cada parte para o todo.
*   **Quando usar:**
    *   Visualizar a proporção de diferentes categorias em um conjunto de dados.
    *   Mostrar a composição de um todo.
*   **Exemplo:** Mostrar a distribuição de clientes por região.

### 12. Mapa de Árvore (Treemap)

*   **Descrição:** Exibe dados em retângulos aninhados, onde o tamanho de cada retângulo representa o valor correspondente.
*   **Quando usar:**
    *   Visualizar a hierarquia de dados e a proporção de cada categoria.
    *   Mostrar a composição de um todo com múltiplos níveis de detalhe.
*   **Exemplo:** Mostrar a estrutura organizacional de uma empresa, com o tamanho de cada departamento representando o número de funcionários.

## Outros Gráficos

### 13. Cartão (Card)

*   **Descrição:** Exibe um único valor.
*   **Quando usar:**
    *   Destacar uma métrica importante.
    *   Mostrar um valor chave.
*   **Exemplo:** Mostrar o total de vendas.

### 14. Medidor (Gauge)

*   **Descrição:** Exibe um valor em relação a uma meta.
*   **Quando usar:**
    *   Visualizar o progresso em direção a uma meta.
    *   Mostrar o desempenho em relação a um objetivo.
*   **Exemplo:** Mostrar o progresso em direção a uma meta de vendas mensal.

### 15. Mapa (Map)

*   **Descrição:** Exibe dados em um mapa geográfico.
*   **Quando usar:**
    *   Visualizar dados relacionados a localizações geográficas.
    *   Mostrar a distribuição de dados em um mapa.
*   **Exemplo:** Mostrar as vendas por estado.

### 16. KPI (Key Performance Indicator)

*   **Descrição:** Exibe um valor, uma meta e um status.
*   **Quando usar:**
    *   Monitorar o desempenho de uma métrica em relação a uma meta.
    *   Mostrar o status de uma métrica.
*   **Exemplo:** Mostrar a receita mensal, a meta de receita e o status (acima ou abaixo da meta).

### 17. Gráfico de Funil (Funnel Chart)

*   **Descrição:** Exibe um processo sequencial, mostrando a redução de valores em cada etapa.
*   **Quando usar:**
    *   Visualizar um processo de conversão.
    *   Identificar gargalos em um processo.
*   **Exemplo:** Mostrar o processo de vendas, desde o contato inicial até o fechamento do negócio.

### 18. Tabela (Table)

*   **Descrição:** Exibe dados em linhas e colunas.
*   **Quando usar:**
    *   Mostrar dados detalhados.
    *   Permitir a ordenação e filtragem de dados.
*   **Exemplo:** Exibir uma lista de clientes com seus respectivos dados de contato e histórico de compras.

### 19. Matriz (Matrix)

*   **Descrição:** Similar à tabela, mas com a possibilidade de agrupar dados em linhas e colunas.
*   **Quando usar:**
    *   Mostrar dados agregados em duas dimensões.
    *   Realizar análises de drill-down.
*   **Exemplo:** Mostrar as vendas por produto e por região.

## Considerações Adicionais

*   **Público:** Considere o público-alvo ao escolher um gráfico. Gráficos mais simples podem ser mais fáceis de entender para um público geral.
*   **Objetivo:** Determine qual é o objetivo principal da visualização. Escolha um gráfico que destaque a informação mais importante.
*   **Dados:** Analise os dados disponíveis e escolha um gráfico que se adapte ao tipo e quantidade de dados.
*   **Teste:** Experimente diferentes tipos de gráficos para ver qual deles comunica melhor a informação.
